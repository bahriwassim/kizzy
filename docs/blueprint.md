# **App Name**: Kizzy 2K26 night

## Core Features:

- Seat Selection: Interactive seat selection on a venue map before booking.
- Ticket Purchase: Secure and streamlined process for purchasing tickets.
- Backoffice: Real-time backoffice with user and permission management for administrators to manage seats, tickets, prices and availability
- Promotional Codes: Management of promotional codes, including percentage or fixed amount discounts.
- Analytics Dashboard: Dashboard for event organizers to monitor sales trends, popular sections, and overall revenue with export features.

## Style Guidelines:

- Primary color: Midnight blue (#191970) for a sophisticated and celebratory feel.
- Background color: Dark grey (#282828) for a modern and elegant look.
- Accent color: Gold (#FFD700) to represent festivity and exclusivity.
- Body and headline font: 'Belleza', sans-serif, for headlines and short amounts of text; with 'Alegreya', serif, for longer body text.
- Use elegant, minimalistic icons that symbolize new year's celebrations and ticketing functions.
